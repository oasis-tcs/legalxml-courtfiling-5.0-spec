Attribute queryError

A query error should be generated when: * The query arguments are considered to be invalid by the receiver, and the query cannot be processed. * The query represents a request for a specific data item, which cannot be retrieved. (another flavor of invalid argument) * The receiving system explicitly fails while processing the query.
